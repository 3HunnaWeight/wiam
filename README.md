# Loans API

API для подачи и обработки заявок на займ.

---

## Запуск проекта

1. Клонировать репозиторий:
git clone https://github.com/3HunnaWeight/wiam.git

3. Поднять окружение
docker-compose up -d --build

---
После запуска сервер приложения будет доступен по адресу: http://localhost:80

## 1. Подача заявки на займ

**POST** `/requests`  

**Описание:** Подать новую заявку на займ. Заявка сохраняется в базе данных.  

**Тело запроса:**

```json
{
  "user_id": 1,
  "amount": 3000,
  "term": 30
}
```
## 2. Обработка заявок

**GET** `/processor?delay=<seconds>`  

**Описание:** Запускает обработку заявок на займ.

**Параметры запроса:**

```json
{
  "delay": 5
}
```
